基本图标



使用样式类fa可让图标字体的代码展示为图标。 Font Awesome是为使用内联元素而设计的。我们通常更喜欢使用 <i> ，因为它更简洁。 但实际上使用 <span> 才能更加语义化。 

<i class="fa">&#xf004;</i>

额外引入css后，可以使用 fa-* icon-* 类来使用图标，不需要使用代码。

<i class="fa fa-heart"></i>
<i class="icon-heart"></i>

Font Awesome 图标字体的代码列表

大图标

    

使用 fa-lg (33%递增)、fa-2x、 fa-3x、fa-4x，或者 fa-5x 类 来放大图标。 

<i class="fa fa-lg">&#xf083;</i>
<i class="fa fa-2x">&#xf083;</i>
<i class="fa fa-3x">&#xf083;</i>
<i class="fa fa-4x">&#xf083;</i>
<i class="fa fa-5x">&#xf083;</i>

固定宽度

 Home  Library  Applications  Settings

使用 fa-fw 可以将图标设置为一个固定宽度。主要用于不同宽度图标无法对齐的情况。 尤其在列表或导航时起到重要作用。 

<div class="list-group">
	<a class="list-group-item" href="#"><i class="fa fa-fw">&#xf015;</i> Home</a>
	<a class="list-group-item" href="#"><i class="fa fa-fw">&#xf02d;</i> Library</a>
	<a class="list-group-item" href="#"><i class="fa fa-fw">&#xf040;</i> Applications</a>
	<a class="list-group-item" href="#"><i class="fa fa-fw">&#xf013;</i> Settings</a>
</div>

动画

    

使用 fa-spin 类来使任意图标旋转，现在您还可以使用 fa-pulse 来使其进行8方位旋转。尤其适合   。 

<i class="fa fa-spin">&#xf110;</i>
<i class="fa fa-spin">&#xf1ce;</i>
<i class="fa fa-spin">&#xf013;</i>
<i class="fa fa-spin">&#xf021;</i>
<i class="fa fa-pulse">&#xf110;</i>

 CSS3动画不支持IE9以下。

<i class="fa fa-rotate-180">&#xf132;</i>

<i class="fa fa-flip-vertical">&#xf132;</i>

组合使用

  fa-twitter on fa-square-o
  fa-flag on fa-circle
  fa-terminal on fa-square
  fa-ban on fa-camera

如果想要将多个图标组合起来，使用 fa-stack 类作为父容器， fa-stack-1x 作为正常比例的图标， fa-stack-2x 作为大一些的图标。还可以使用 fa-inverse 类来切换图标颜色。您可以在父容器中 通过添加 大图标 类来控制整体大小。 

<span class="fa-stack fa-lg">
	<i class="fa fa-stack-2x">&#xf096;</i>
	<i class="fa fa-stack-1x">&#xf099;</i>
</span>
fa-twitter on fa-square-o<br>
<span class="fa-stack fa-lg">

	<i class="fa fa-stack-1x fa-inverse">&#xf024;</i>
</span>

<span class="fa-stack fa-lg">

</span>

图标的可访问性

使用 aria-hidden 可以禁止屏幕朗读工具作用。使用 aria-label可以让屏幕朗读工具读指定的东西。 

<button type="button" class="btn btn-default" aria-label="Left Align">
	<i class="fa" aria-hidden="true">&#xf036;</i>
</button>
<button type="button" class="btn btn-default btn-lg">
	<i class="fa" aria-hidden="true">&#xf005;</i> Star
</button>
