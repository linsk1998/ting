字体

类 描述
.font-sans 非衬线字体(黑)，通常在标题使用。
.font-serif 衬线字体(宋)，通常在正文使用。
.font-cursive 手写体(楷)，通常在横幅使用。
.font-monospace 等宽字体，通常在代码使用。
