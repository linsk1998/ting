助手

清除浮动

类 描述
.pull-left 元素浮动到左边
.pull-right 元素浮动到右边
.clearfix 清除容器内的浮动浮动

例子

视觉隐藏

通过 .visually-hidden 类在视觉上隐藏某个元素的同时，仍然能够被辅助技术识别。

类 描述
.visually-hidden 在视觉上隐藏某个元素。

显示

类 描述
.hidden-desktop 在电脑上隐藏隐藏
.hidden-mobile 在手机或平板上隐藏
.hidden-phone 在手机隐藏
.hidden-tablet 在平板隐藏
.hidden-lg 在大屏幕隐藏
.hidden-md 在中隐藏
.hidden-sm 在略小隐藏
.hidden-xs 在小隐藏
.hidden 强制元素隐藏
.block 将元素显示为block
.inline-block 将元素显示为inline-block
.inline 将元素显示为inline

文本截断

类 描述
.line-{n} 强制高度为n行
.line-max-{n} 高度不得超过n行

图标

类 描述
.icon 表示一个图标，生成一个方形inline-block。
.icon-fw 可以将图标设置为一个固定宽度。
.icon-lg .icon-2x .icon-3x .icon-4x 大字体图标。
.icon-rotate-* .icon-flip-* 对图标进行旋转和翻转

动画

类 描述

.animated.infinite 循环动画。
.animated.fast 快速。
.animated.slow 慢速。
